# 🤖 Machine Learning – Newcomer Level

Selamat datang di level **Newcomer** untuk jalur Machine Learning! Di level ini, kamu akan mengenal dasar-dasar machine learning, algoritma sederhana, dan konsep fundamental dalam AI.

---

## 🎯 Tujuan Level

- Mengenal konsep dasar machine learning dan AI
- Belajar algoritma ML sederhana (linear regression, classification)
- Memahami data preprocessing dan feature engineering
- Melakukan model evaluation dasar
- Mengembangkan mindset ML dan AI

---

## 📁 Daftar Exercise

| No. | Judul Latihan                        | Fokus                                     |
|-----|--------------------------------------|-------------------------------------------|
| 01  | Konsep Dasar Machine Learning        | Supervised vs unsupervised, overfitting   |
| 02  | Linear Regression Sederhana          | Regression, model training, evaluation   |
